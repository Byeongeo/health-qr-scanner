# QR 점수 스캐너 (보건교육)

학생이 자기 **학번 QR**을 켜진 카메라에 대면, 학번↔이름을 매칭해 **점수를 누적**하고
**비프음 + 음성 안내 + 큰 화면 표시**를 해 주는 키오스크형 웹앱.

카메라가 막히는 Apps Script 화면 대신 **Vercel(Next.js)** 에서 화면을 띄우고,
데이터 처리는 시트에 붙여넣은 **Apps Script**(= 백엔드)가 맡고, 저장은 **Google Sheets**.

## 구조
```
브라우저(Vercel 카메라 페이지)
  │ 학번 전송
  ▼
Vercel API (얇은 중계: app/api/scan·config)   ← 환경변수 APPS_SCRIPT_URL 1줄
  │ 서버끼리 호출(CORS 없음)
  ▼
Apps Script (시트에 붙여넣은 Code.gs = 백엔드)  ← 이름조회·점수누적·멘트선택
  ▼
Google Sheets (탭 5개)
```

## 동작 흐름
```
풋마우스(밟기=클릭/Space) → 카메라 ON → QR(학번) 인식
  → Apps Script 가 이름 조회·+점수 누적·멘트 선택
  → 비프음 + 음성 2문장 + 큰 화면 표시 → 카메라 OFF
```

## 설치 — 교사용 (Google Sheets 쪽, 최초 1회)

1. **점수용 구글시트**를 새로 만든다(빈 시트면 됨).
2. [확장 프로그램] → [Apps Script] 열기.
3. [apps-script/Code.gs](apps-script/Code.gs) 내용 전체를 복사해 붙여넣고 저장.
4. 함수 목록에서 **`setup` 실행** → 권한 승인 → **탭 5개·헤더·예시·QR 수식 자동 생성**.
   (또는 시트를 새로고침하면 상단 [QR 점수 스캐너] 메뉴 → 초기 설정)
5. [배포] → [새 배포] → 유형 **"웹 앱"**
   - 실행: **나(소유자)** / 액세스 권한: **모든 사용자** → 배포 → **웹앱 URL 복사**(`…/exec`).
6. (권장) 보안: `setSecret` 의 값을 원하는 비밀키로 바꿔 1회 실행.

> 학생 명단은 `student list` 탭에 학번·이름을 입력하면 C열에 QR이 자동 생성됩니다. 인쇄해 배부.

## 설치 — 배포 쪽 (Vercel, 운영자 1회)

환경변수 ([.env.example](.env.example) 참고):
- `APPS_SCRIPT_URL` = 위 5번에서 복사한 웹앱 URL
- `APP_SHARED_SECRET` = 위 6번 비밀키 (안 썼으면 비워둠)

```bash
npm install
cp .env.example .env.local   # 값 채우기
npm run dev                  # http://localhost:3000  (손마우스로 바로 테스트)
npm run build                # 배포 전 검증
```

GitHub 푸시 → Vercel 연결 → 환경변수 입력 → 배포. 브라우저에서 카메라 권한 1회 허용하면 끝.

## 설정값 (시트 `config` 탭에서 교사가 수정)
| 키 | 기본값 | 설명 |
|---|---|---|
| 회당가점 | 10 | 1회 스캔 점수 (바꾸면 다음 스캔부터 적용, 과거 점수 유지) |
| 쿨다운초 | 0 | 같은 학생 재적립 방지(0=제한없음) |
| 카메라타임아웃초 | 8 | 밟은 뒤 자동 꺼짐 |
| 트리거키 | Space | 풋마우스가 보내는 키(빈칸=아무 클릭) |
| 첫문장 | {이름} 학생, {추가}점이 가산되어 총 {총점}점입니다 | 음성·화면 1줄 |
| 미등록처리 | 거부 | 거부 / 허용 |
| 표시시간초 | 4 | 결과 화면 유지 시간 |

`messages` 탭 — 점수대별 멘트(치환변수 `{이름} {학번} {추가} {총점}`):
| 임계점수 | 멘트 | 1회만 |
|---|---|---|
| 0 | {이름} 학생 힘내세요 | N (상시) |
| 50 | {이름} 학생 잘했어요 | N (상시) |
| 100 | 선생님에게 선물을 받아가세요 | Y (처음 넘을 때 1회) |
| 150 | 하트스티커 3개를 받아가세요 | Y (처음 넘을 때 1회) |

## 풋마우스
페달이 보내는 신호(클릭 또는 특정 키)를 `config.트리거키` 에 입력.
빈칸이면 "아무 클릭"으로 작동(손마우스 테스트용).
